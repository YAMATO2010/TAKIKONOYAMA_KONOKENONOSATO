
# TAKIKONOYAMA_KONOKENONOSATO

「きのこたけのこ戦争」における、名前の順序がもたらす不平等を解消するために開発されました。

## 開発の動機
「きのこたけのこ戦争」という言葉において、常に「きのこ」が先に来る現状。
これは「きのこ」が優位であるかのような誤解を与えかねず、たけのこ派への事実上の宣戦布告とも取れます。

この火種を消し去り、世界を真の平和へと導くため、私は「きのこたけのこ」をランダムに並び替え、その概念をぐちゃぐちゃにするアプリを開発いたしました。

## 特徴
- **完全な平等**: 実行するたびに順序が入れ替わり、どちらが上という概念を破壊します。
- **平和への祈り**: このアプリの実行により、世界はまた一歩平和へと近づきました。
- **圧倒的ビジュアル**: 背景にはAI（Gemini）によって生成された、全てを過去にするムキムキの守護神が君臨しています。
